Vi-SakiWin64

すべてのvi使いへの、近道。コンパイルを通る詩。

雨の午後、静かにキーを叩いて――ファイルを、守りたかったの。

由来

見つけられる最後のソースは Traditional Vi でした。オリジナルの vi は、そのアカウントが消えたことで消えてしまいました。 フォークの一つさえ残されていない——まるで、雨に溶けた手紙のように。

袁志豪 / lichray さんが最後まで守ってくれたことに感謝しています。 Keith Bostic氏の Nvi は依存関係が多すぎて、 修正するのは廃墟の中で図書館を再建するようなものでした。 そのため、ソースコードは busybox-w32 から直接取得しました——まだ燃えている小さな炎。

このバージョンは Ron Yorston 氏のコミットに基づいています：
ash: fix execution of applets via Unix-style path
686a0803f974

感謝：Bill Joy（創造者）、Gunnar Ritter（守護者）、Ron Yorston（現門番）。

機能

🌏 三言語インターフェース
繁體中文 / 日本語 / English
📦 単一実行ファイル
一つの .exe、PATH に入れるだけ
🪟 ネイティブ Win64
WriteConsoleW で Unicode を直接出力、文字化けなし
📖 :help
三言語クイックリファレンス内蔵

インストール

# Scoop（推奨）
scoop bucket add sakivi https://github.com/Saki-tw/scoop-sakivi
scoop install saki-vi

# Winget
winget install SakiStudio.SakiVI

# 手動：Releases からvi.exe をダウンロード、PATH に配置

使い方

vi file.txt          # ファイルを編集
vi -H                # 有効な機能を表示
vi --version         # バージョン情報
:help                # 三言語リファレンス
:features            # ビルド機能一覧

ビルド

MSYS2 UCRT64 環境：

git clone --recursive https://github.com/Saki-tw/Vi-SakiWin64.git
cd Vi-SakiWin64
bash scripts/build.sh
