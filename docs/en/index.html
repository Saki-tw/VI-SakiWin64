Install

# Scoop (recommended)
scoop bucket add sakivi https://github.com/Saki-tw/scoop-sakivi
scoop install saki-vi

# Winget
winget install SakiStudio.SakiVI

# Manual: download vi.exe from Releases, place in PATH

Usage

vi file.txt          # edit file
vi -H                # show enabled features
vi --version         # version info
:help                # trilingual reference
:features            # compiled features

Build

Requires MSYS2 UCRT64:

git clone --recursive https://github.com/Saki-tw/Vi-SakiWin64.git
cd Vi-SakiWin64
bash scripts/build.sh
