Vi-SakiWin64

給所有 vi 派的高速捷徑。一個能通過編譯的詩。

人間別久不成悲——但至少 :wq 還能存檔。

淵源

由於能找到的最後一個來源是 Traditional Vi， 原始 vi 的版本已經因為帳號消失而消失，就連一個 Fork 都沒留下—— 那些程式碼，像是被撕掉的書頁。

感謝袁志豪 / lichray 將它留存至最後。 Keith Bostic 所創之 Nvi 依賴太多東西，修改起來如同在廢墟中重建一座圖書館。 因此源碼直接取自 busybox-w32， 一個仍在燃燒的火種。

此版本基於 Ron Yorston 的提交：
ash: fix execution of applets via Unix-style path
686a0803f974

感謝：Bill Joy（創世者）、Gunnar Ritter（守護者）、Ron Yorston（現任守門人）。

特色

🌏 三語介面
繁體中文 / 日本語 / English
📦 單檔免安裝
一個 .exe，放進 PATH 就能用
🪟 原生 Win64
WriteConsoleW 直接輸出 Unicode，不怕 Big5 亂碼
📖 :help
內建三語快速參考卡

安裝

# Scoop（推薦）
scoop bucket add sakivi https://github.com/Saki-tw/scoop-sakivi
scoop install saki-vi

# Winget
winget install SakiStudio.SakiVI

# 手動：從 Releases 下載 vi.exe，放入 PATH

使用

vi file.txt          # 編輯檔案
vi -H                # 顯示啟用功能
vi --version         # 版本資訊
:help                # 三語快速參考
:features            # 編譯功能一覽

建置

MSYS2 UCRT64 環境：

git clone --recursive https://github.com/Saki-tw/Vi-SakiWin64.git
cd Vi-SakiWin64
bash scripts/build.sh
